BIM Georeferencing Audit Script (ArcPy)
A Python script that scans a folder of BIM files (.rvt, .ifc) and generates a QA report summarizing spatial reference (CRS), exterior extents, and georeferencing status.
The script identifies un‑georeferenced models, reports recorded extents from the ExteriorShell, and produces a timestamped, human‑readable text report suitable for BIM intake validation, GIS integration workflows, and vendor quality checks.
